# iBaiRelease

**One app for all your self-hosted media.**

iBai is a native Apple-platform client for music, audiobooks, and video from your own servers. Connect to Navidrome, Audiobookshelf, Emby, Plex, WebDAV, SMB, Google Drive, Dropbox, and more — all in one unified SwiftUI app for iPhone, iPad, Mac, and Apple TV.

## Features

### Music

- **Navidrome** — full Subsonic API support for your self-hosted music library
- Browse playlists, artists, albums, and songs
- **Daily Mix**, **Genre Mix**, and radio-style recommendations
- Full-screen player with synced **lyrics**, similar-song suggestions, and favorites
- Play queue with history, saved queues, and shuffle / repeat modes
- **Offline music cache** for playback without a network
- **Scrobbling** to [Last.fm](https://www.last.fm/) and [ListenBrainz](https://listenbrainz.org/)
- Lock screen, Control Center, and **CarPlay** remote controls

### Audiobooks

- **Audiobookshelf** integration
- Library browsing and search
- **Offline download** with progress tracking
- Shared audio player with music playback

### Video

- **Emby**, **Plex**, **WebDAV**, **SMB**, **Google Drive**, and **Dropbox**
- Multiple server profiles with per-server authentication
- Discovery home: Continue Watching, Recently Added, Recommendations, and more
- Series and season navigation with resume playback
- **Offline video downloads**
- Powered by **FFmpeg** for broad format support
- Built-in local HTTP proxy for efficient streaming from remote sources

### General

- Native UI for **iOS**, **iPadOS**, **macOS**, and **tvOS**
- Light / Dark / System theme
- In-app language: English, Simplified Chinese, Traditional Chinese (HK & TW)
- Image, music, audiobook, and video cache management
- Credentials stored in the system **Keychain**
